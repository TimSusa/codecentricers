# Quickstart

## Installation
```
  npm install
  bower install (optional)
```

## Serve Project:

```
  grunt serve --variant=generic --flavor=develop
```

## Test
```
  grunt test --variant=generic --flavor=develop
```

## CI Build:

```
  grunt cibuild --variant=generic --flavor=develop

  or

  make use of the build-frontend.sh script
```
