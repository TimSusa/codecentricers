timeoutInterceptor
property in module ng

Description

$http interceptor that registeres a dynamic timeout for each API call.

Dependencies

  • $timeout
  • $log
  • $q
  • globals

Usage

timeoutInterceptor

Example

angular.module( 'codecentricers', []).config(['$httpProvider', function ($httpProvider) {
  $httpProvider.interceptors.push('timeoutInterceptor');
}]);
