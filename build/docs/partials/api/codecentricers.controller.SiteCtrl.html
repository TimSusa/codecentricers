SiteCtrl
controller in module codecentricers

Description

Controller for global site settings used in index.html. Removes the no-js class from the <html> tag to confirm Angular has started up correctly.

The site controller exposes the following objects

  • appVersion - {object} - application-wide build info
  • customer - {object} - customer object from global configuration
  • features - {object} - feature object from global configuration

Dependencies

  • $interpolate
  • $translate
  • $document
  • $location
  • $route
  • authService
  • version

Methods

  • getSiteClasses()

    Returns an array of CSS classes to be set on the document's body.

    Returns
    Array

    Array of CSS class names to be placed on the <body> tag.

Example

<html lang="{{site.getLang()}}" class="no-js" ng-app="codecentricers"
  ng-controller="SiteCtrl as site" ng-class="site.getSiteClasses()">
<head>
    <base href="/">
    <script type="text/javascript" src="scripts/codecentricers.min.js"></script>
</head>
<body>
</body>
</html>
