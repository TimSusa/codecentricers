MainPageCtrl
controller in module codecentricers

Description

This controller is supposed to fetch member data via Github resource. Furthermore, the fetched information can be stored in local browser storage. In this manner the fetched information can be used for offline purposes.

Dependencies

  • storageService
  • Github

Methods

  • beautifyLanguageInfo()

    Beautifies the object response, to have languages and their written bytes lenght availlable.

  • handleMemberReposSuccess()

    On successful response, this function will handle and sort the data and furthermore request for the used languages from that repo.

  • handleQueryMembersSuccess()

    On successful response, this function will handle and sort member data.

  • load()

    Stores the blob to webstorage.

  • load()

    Loads the blob from webstorage.

  • printError()

    On not successful response, this function will just print the error.

  • queryAllMemberRepos()

    Will query all repositories for all members
