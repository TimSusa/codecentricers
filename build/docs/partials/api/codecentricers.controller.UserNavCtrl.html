Methods

  • is(segment, pos)

    Check the current URL path obtained from $location matches a segment. This is useful to set CSS active class on navigation elements.

    Parameters
    ParamTypeDetails
    segmentString

    String to compare against a url segment.

    posNumber

    Position in the URL path to compare, default: 1, When 0 matches segment against the entire URL path.

    Returns
    boolean

    True if the current route contains the string.
